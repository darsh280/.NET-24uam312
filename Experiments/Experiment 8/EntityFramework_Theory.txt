Entity framework is a tool in .NET that help us work with database using C# code instead of Sql queries.
Why need Entity (Advantages)
    - less code
    - fast development
    - easy to maintain

when we use sql - time consuming
                - error chances are high
                - highly complex

Types - Entity framework vs Entity framework core
        EF older version - heavy - only for windows
        EF core advanced version  - light weight - cross platform working(windows, macos, linux)

What is ORM(Object relational mapping)
- It mapps C# classes to database tables and object to row
- Example 
public class Student{
    public string name;
    public int id {get;set;}
    public Name{
        get{return name;}

}
